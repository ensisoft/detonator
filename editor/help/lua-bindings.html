Scene

Scene is the basic representation of the game world. It contains the entities arranged in a hierarchy. The scene is initially built in the editor application and during the game play you can access an instance of the scene. When the game play with some scene begins the scene represenation built in the editor is used to create an instance and all scene nodes placed in the world in the editor are used to create entity instances. While the game runs you can interact with the scene, by example querying it for entities and spawning new entities inside it.

Entity

Entities are the objects that are spawned in the game. When the game play beings for each object that is placed in the scene a new entity is spaned automatically. When the game play is going on it's possible to alter the game world by adding/spawning new entities and removing/deleting existing entities. It's also possible to alter the hierarchy of the game world by linking entities with other entities thus forming transformation hierarchies.

Methods

GetTranslation
Returns the entity's translation (x, y offset) relative to its parent. If the entity's parent is the game world root then this translation is also the global x,y position of the entity.
